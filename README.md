## 解决的问题

在智能温室或植物生长监测场景中，使用普通单目摄像头进行叶面积估算时，由于设备微小抖动、重新安置或支架位移，相机的拍摄姿态（高度、俯仰角）往往会发生各种细微变化。这种变化会导致显著的透视失真，从而使提取的“像素面积”出现严重波动，使得单纯基于像素的对比失去意义。

本项目旨在通过低成本方案（单目摄像头 + 打印棋盘格），消除由相机姿态变化带来的透视误差，实现抗扰动的植物叶片真实物理面积高精度动态测量。

## 本项目做了什么 (当前进度)

代码核心已经重构并升级为基于 **SAM3 分割 + 多视角几何** 的方法框架。主要完成了以下三部分核心功能：

1. **单应矩阵提取 (Homography Matrix)**
   通过捕捉画面中固定尺寸棋盘格的特征角点，建立图像像素坐标系与真实物理世界坐标系（棋盘格XY平面）之间的单应矩阵映射，打通了像素到绝对物理尺度的投影通道。

2. **多帧/两帧图像视角对齐 (Two-frame Image Alignment)**
   针对因相机扰动或位移产生的空间姿态差异，应用单应矩阵进行透视变换 (`cv2.warpPerspective`)。将由于透视效应产生形变的后续帧，在空间视角上直接“拉平”并重投影还原至参考帧的视角，实现两帧图像物理级别的空间对齐与校正。

3. **鞋带算法求解多边形真实面积 (Shoelace Algorithm)**
   摒弃了通过固定因子换算的面积估算法，转而采用更加严谨的纯几何维度面积计算：
   - 首先使用 SAM3 模型提取出高精度的叶片掩码边缘轮廓像素坐标。
   - 利用单应矩阵，将所有二维像素级的边缘坐标逆变换映射到三维空间中的物理平面坐标系（世界坐标 XY 坐标）。
   - 最后，对于散布在真实物理平面上的多边形轮廓点列，通过迭代遍历应用**鞋带公式 (Shoelace formula，测量师公式)**，精确求解出植物叶片在物理平面的真实绝对面积。

## 运行说明

### 环境准备

推荐使用 Anaconda 创建独立环境：

```bash
conda create -n plant_measure python=3.8
conda activate plant_measure

pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
pip install opencv-python opencv-contrib-python pandas matplotlib pillow
```
